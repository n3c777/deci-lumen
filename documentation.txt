
-Installing Flask
to install flask must create virtual evniorment. Run this command before any pip commmands
python -m venv .venv
.venv\Scripts\activate
if this doesn't work there's a few ways to create the virtual enviorment in python. CNRL + P and then > in the search bar. Then either enter a new path (the base pyhton.exe worked one time for me) or create a virtual enviorment with the current .venv script path (this worked for me the most). To debug flask use: python -m flask run --debugger
To see routes use:  flask routes
